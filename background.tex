Chernoff and Zacks (1964):
\begin{itemize}
\item Constant probability of change
\item Parameter of the new regime only depends on the parameters of the immediately previous regime, i.e. $\theta_k$ is drawn from some distribution $\theta_k \mid \theta_{k-1}$ that depends only on $\theta_{k-1}$
\item The parameters from this distribution (called hyperparameters) are either specified or estimated from the data
\end{itemize}

Yao (1984)
\begin{itemize}
\item Same model, but the joint distribution of the parameters $\{ \theta_k \}$ is exchangeable and independent of the change points.
\end{itemize}
